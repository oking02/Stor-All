Collection Types

Stor-All organises work by dividing in into four catagorises. These are used to seperate and create a usable heirarchy within the data.

Experiments and Analysis

Experiments are the main container of information. It's use is up to the owner but a generally an experiment is a single run of analysis. This could be just a simple analysis run or multiple related ones.

Projects

Projects are a higher level than experiments. Each project must have a specific owner. Project will have Experiments under them. Projects can be used to group related Experiments together.
